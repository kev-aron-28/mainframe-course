# What is DB2?

Is a relational database management system RDBS developed by IBM. IT supports storing, retrieving and managing data using SQL.
It excels in its ability to process massive transactional loads. ITs widely used in enterprise level applications like banking, finance and insurance

# Advantages
- Understanding data in tables is very easy compared to the other data managment methods
- High performance: Suitable for handling millions of transaction simultaneously in mission-critical environments
- Highly secured
- Supports OLTP and OLAP 
- SQL and NoSQL

# Disadvantages
- Cost
- Complexity in setup and maintenence
- Resource intensive
- LImited flexibility in platform migration

# Types of SQL Statement
SQL statements can be categorized into five main types:
- DQL (SELECT)
- DDL (CREATE,ALTER,DROP)
- DML (INSERT, UPDATE, DELETE)
- DCL (GRANT, REVOKE)
- TCL
